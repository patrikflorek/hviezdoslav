## HTML dataset

These files represent most of the poetic work of Slovak writer [Pavol Orszagh Hviezdoslav](https://en.wikipedia.org/wiki/Pavol_Orsz%C3%A1gh_Hviezdoslav). 

Using `download_html.py` script, the data were first downloaded from [Zlaty fond SME](https://zlatyfond.sme.sk/autor/56/Pavol-Orszagh-Hviezdoslav) and stored in `./HTML` folder; each file represents one book.

```
python download_html.py
```
